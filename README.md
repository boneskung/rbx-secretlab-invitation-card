# Invitation Card Tools — SECRET LABS

เครื่องมือเว็บสำหรับ **generate ภาพการ์ดเชิญ (invitation card)** ของคลับ SECRET LABS
**HTML + JavaScript ล้วน ๆ** ไม่มี build step ไม่มี dependency — เปิดในเบราว์เซอร์ได้เลย

🔗 **ใช้งานออนไลน์ (GitHub Pages):** https://boneskung.github.io/rbx-secretlab-invitation-card/

## ทำอะไรได้บ้าง

- **รูปคู่กับชื่อ 3 แบบ** — เลือกประเภทการ์ดได้: `ชื่อคลับ` / `ชื่อเจ้าของคลับ` / `ชื่อผู้เล่นเฉพาะ`
  (ป้ายกำกับช่องกรอกจะเปลี่ยนตามโหมด)
- **2 อัตราส่วน** — แนวตั้ง `1080 × 1350` และ จัตุรัส `1080 × 1080`
- **พื้นหลัง** — เลือกจากแกลเลอรีในตัว หรืออัปโหลดรูปเอง · ลากเพื่อจัดตำแหน่ง · สกอลล์เพื่อซูม
- **โลโก้ SECRET LABS** — วางได้ 9 ตำแหน่ง ปรับขนาด/ความทึบ
- **โลโก้คู่ (เชิญคลับ)** — แนบโลโก้คลับที่เชิญ แล้วเลือกโหมด `คู่ (SECRET LABS × คลับ)`
  จะวางโลโก้แมพ SECRET LABS คู่กับโลโก้คลับ มี `×` คั่นกลาง (ปิด/เปิดได้)
  ส่วนชื่อคลับใส่ในช่อง **ชื่อคลับ** (โหมด ① ประเภทการ์ด = ชื่อคลับ)
- **ข้อความครบ** — หัวเรื่อง, ชื่อหลัก, บรรทัดรอง, วัน-เวลา, สถานที่, footer
- **สไตล์** — 7 ฟอนต์ (รวมฟอนต์ไทย Kanit/Prompt), จัดชิด ซ้าย/กลาง/ขวา,
  วางข้อความ บน/กลาง/ล่าง, ปรับขนาดชื่อ
- **ธีมสำเร็จรูป + เอฟเฟกต์** — Gold Noir / Neon Red / Silver Chrome / Royal Purple / Emerald,
  เงาดำปรับได้ 4 ทิศ, กรอบขอบ, เกรนฟิล์ม
- **เทมเพลต** — บันทึก/โหลด/ลบ ชุดตั้งค่าไว้ใช้ในอนาคต (เก็บใน browser)
  พร้อม **ส่งออก/นำเข้า JSON** เพื่อย้ายเครื่องหรือแชร์
- **ดาวน์โหลด PNG** — เรนเดอร์ความละเอียด 2× คมชัด

## วิธีใช้

แค่เปิด URL ของ GitHub Pages ด้านบน — ไม่ต้องติดตั้งอะไร

### รันในเครื่อง (local)

วิธีง่ายสุด: ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ได้เลย

> ⚠️ ถ้าเปิดแบบ `file://` แล้วปุ่ม **ดาวน์โหลด PNG** ใช้ไม่ได้ (บางเบราว์เซอร์บล็อก canvas)
> ให้เสิร์ฟผ่าน static server ตัวไหนก็ได้ เช่น:
> ```bash
> python -m http.server 4321      # แล้วเปิด http://localhost:4321
> # หรือ
> npx serve .
> ```
> บน GitHub Pages (https) ทำงานครบทุกฟีเจอร์อยู่แล้ว

## โครงสร้าง

```
rbx-secretlab-invitation-card/
├─ index.html          # หน้าเครื่องมือ
├─ app.js              # เอนจิน canvas + เทมเพลต
├─ style.css           # ธีมดำหรู SECRET LABS
├─ assets-manifest.js  # รายการ asset (โลโก้ + แกลเลอรีพื้นหลัง)
├─ assets/
│  ├─ logo.png             # โลโก้ SECRET LABS
│  └─ screenshots/         # รูปพื้นหลังในแกลเลอรี
│     ├─ map-01.png …
└─ .nojekyll          # บอก GitHub Pages ไม่ต้องรัน Jekyll
```

## เพิ่มรูปพื้นหลัง / เปลี่ยนโลโก้

1. ก๊อปไฟล์รูปลง `assets/screenshots/`
2. เพิ่มอีกหนึ่งบรรทัดใน `assets-manifest.js`:
   ```js
   { name: "map-04", url: "assets/screenshots/map-04.png" },
   ```
3. commit + push — Pages จะอัปเดตให้เอง

เปลี่ยนโลโก้ SECRET LABS: ทับไฟล์ `assets/logo.png`

## หมายเหตุ

- เทมเพลตเก็บใน `localStorage` ของเบราว์เซอร์ — ใช้ปุ่ม **ส่งออก JSON** ถ้าอยากสำรอง/ย้ายเครื่อง
  (โลโก้คลับที่แนบจะถูกบันทึกติดไปกับเทมเพลตด้วย)
- ฟอนต์ดึงจาก Google Fonts (ต้องต่อเน็ตครั้งแรก) ถ้าออฟไลน์จะ fallback เป็นฟอนต์ระบบ
